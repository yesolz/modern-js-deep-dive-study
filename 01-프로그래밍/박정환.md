# 1. 프로그래밍

## 커뮤니케이션이다.

프로그래밍에 앞서 해결해야 할 문제를 명확히 이해

적절한 문제 해결 방안이 필요하다.

[**문제 해결 능력 요구**]

- 복잡함 ⇒ 단순하게 분해
- 자료 ⇒ 정리,구분
- 순서에 맞게 배열

[**정확하고 상세하게 요구사항을 설명**]

## 1.2 프로그래밍 언어

### 기계어로 명령은 전달해야 한다.

프로그래밍 언어는 **구문**과 **의미**의 조합으로 표현된다.

- **문법** : 약속된 구문
- **컴파일러, 인터프리터** : 컴퓨터가 이해할수 있는 기계어로 번역
